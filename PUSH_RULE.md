# プッシュ前の実行テスト・単体テストルール

## 1. 実行テスト
- プッシュ前に必ず `python app.py` でアプリが正常起動することを確認してください。
- エラーや例外が出ていないことを確認してください。

## 2. 単体テスト
- プッシュ前に必ず `pytest` でテストを実行し、全てのテストがパスすることを確認してください。
- テストファイルは `tests/` ディレクトリにあります。
- テストコマンド例：

```bash
pytest
```

## 3. 注意事項
- テストに失敗した場合は、修正して再度テストを実行してください。
- テスト・実行確認ができていない場合はプッシュ禁止です。

---
このルールは全員厳守してください。
